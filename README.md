### 2. Weather Trek
A weather application providing real-time weather data and forecasts with an intuitive user interface.

**Features:**
- Real-time weather data from external API
- Location-based weather search
- 7-day forecast display
- Responsive design for all devices
- User-friendly interface with weather visualizations

**Tech Stack:**
- **Frontend:** React
- **API Integration:** OpenWeatherMap API
- **Styling:** CSS3 

**Key Highlights:**
- Integrated third-party weather API with error handling
- Implemented geolocation features for automatic location detection
- Created dynamic UI that updates based on weather conditions
- Optimized API calls to reduce unnecessary requests
